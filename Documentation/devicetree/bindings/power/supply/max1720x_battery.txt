MAX17201/MAX17205 Fuel Gauge
----------------------------

Required properties :
 - compatible : "maxim,max1720x"

Optional properties :
 - maxim,fg-config: array of bytes to be uploaded to the shadow RAM to either
   use this config or burn it to the non volatile area.
 - maxim,force-load-config: load the maxim,fg-config data into shadow RAM and
   run the fuel gauge using this config instead of the one from the NV RAM.
 - maxim,force-flash-config: program the loaded config (maxim,fg-config) into
   the non volatile if the this flag is set AND nVEmpty are not matching. Number
   NV block writes are limited to 7 so use with care.
 - maxim,psy-type-unknown: if set, forces poser supply type to be
   POWER_SUPPLY_TYPE_UNKNOWN rather than POWER_SUPPLY_TYPE_BATTERY
